PCA算法步骤
对于m个n维样本
1. 求解出协方差矩阵
2. 求出协方差矩阵的特征值以及对应的特征向量
3. 将特征值按照从大到小排列 并取前k个组成矩阵U
4. Y = UX 是降维后的数据
注意 矩阵U比较重要在测试集上 也需要先用U对测试样本进行降维
如果记整个特征值的和为s, 前k个特征值和为s0 那么s0/s 称为explained variance 
k越大保留的信息越多
explained_variance_ratio_ 是sklearn提供的pca的属性 可以知道当前保留了多少信息


LDA线性判别式分析
一个通俗的理解就是 通过一个投影矩阵 使得两个类(假设我们的样本都是来自两个类)之间分的很开
但是类内又靠的比较近，那么这样就有利于做出分类的选择 这个投影矩阵抓住了本质特征
所以我们会定义类内离散度和类间离散度

LDA算法步骤:
输入: 数据集D={(x1,y1),(x2,y2),...,((xm,ym))},其中任意样本xi为n维向量，yi∈{C1,C2,...,Ck}，降维到的维度d。
输出: 降维后的样本集$D′$
1. 计算类内散度矩阵Sw
2. 计算类间散度矩阵Sb
3. 计算矩阵(Sw)^-1Sb
4. 计算(Sw)^-1Sb的最大的d个特征值和对应的d个特征向量(w1,w2,...wd),得到投影矩阵[Math Processing Error]
5. 对样本集中的每一个样本特征xi,转化为新的样本zi=WTxi
6. 得到输出样本集D′={(z1,y1),(z2,y2),...,((zm,ym))}
